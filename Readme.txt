To run the project

1. Install the amazon alexa, onto the raspberry pi, follow <https://developer.amazon.com/docs/alexa-voice-service/set-up-raspberry-pi.html> 
2. Create IFTTT account and create applets, follow setups from report
3. Create adafruit.io account and similarly follow steps from report to setup the broker account.
4. Burn the NodeMCU code onto the ESP module.
5. Form the circuit.
6. Run alexa on raspberry pi using sudo bash startsample.sh
6. Talk with alexa
	ex: Say "Alexa Trigger" then ask the commands set on IFTTT
